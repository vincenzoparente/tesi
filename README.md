# Tesi
Micro-framework creato per tesi 

Per utilizzare il progetto basta creare un programma .py che svolga una qualsiasi mansione, inserire tale progetto in una cartella separata, nel main del framework alla voce:

if name == "main": avvia_framework_tesi("INSERIRE NOME CARTELLA")

Una volta avviato il programma restituirà un pdf chiamato: "Report_analisi_qualita.pdf", esso conterrà tutte le infomazioni riguardanti le problematiche, insieme ad un punteggio finale in 100esimi

Come esempio per testare ho aggiunto un programma chiamato medio che resitutirà secondo i parametri una valutazione nella media
